# 国际化

ant-design-hmfw 内置中文（`zhCN`）和英文（`enUS`）两种语言包，通过 `ConfigProvider` 的 `locale` 属性切换。

## 使用语言包

```vue
<template>
  <ConfigProvider :locale="locale">
    <App />
  </ConfigProvider>
</template>

<script setup lang="ts">
import { ref } from 'vue'
import { ConfigProvider, zhCN, enUS } from 'ant-design-hmfw'

const locale = ref(zhCN)

const toggleLocale = () => {
  locale.value = locale.value === zhCN ? enUS : zhCN
}
</script>
```

## 内置语言包

| 语言包 | 语言     | 导入路径                                 |
| ------ | -------- | ---------------------------------------- |
| `zhCN` | 简体中文 | `import { zhCN } from 'ant-design-hmfw'` |
| `enUS` | English  | `import { enUS } from 'ant-design-hmfw'` |

## 语言包内容

语言包包含以下组件的文案：

- **Pagination**：上一页、下一页、跳转等
- **DatePicker / TimePicker**：确定、今天、此刻等
- **Modal**：确定、取消等
- **Popconfirm**：确定、取消等
- **Table**：筛选、重置等
- **Upload**：上传文件等
- **Empty**：暂无数据等

## 自定义语言包

可以基于内置语言包扩展或完全自定义：

```ts
import { zhCN } from 'ant-design-hmfw'

const myLocale = {
  ...zhCN,
  Pagination: {
    ...zhCN.Pagination,
    prev_page: '上一页',
    next_page: '下一页',
  },
}
```
