# v0.27.0

发布日期：2026-07-24

## 🔧 重构

### Calendar

- **类型规范化**：使用 `satisfies Record<keyof CalendarProps, any>` 模式确保运行时 props 与 TypeScript 接口完全一致
- **样式 Token 化**：将硬编码样式值迁移到 CSS 变量系统
- **Demo 扩充**：新增自定义 Header、自定义日期单元格等演示

### TreeSelect

- **类型约束强化**：规范 props 定义，消除类型不一致风险
- **样式 Token 化**：迁移到统一的设计 Token 系统
- **无障碍支持增强**：改进键盘导航和 ARIA 属性

### Transfer

- **类型约束强化**：使用 `satisfies` 模式规范 props 定义
- **文档完善**：补充使用说明和最佳实践

### TimePicker

- **类型约束强化**：规范 props 类型定义
- **样式 Token 化**：统一使用 CSS 变量控制样式
- **Demo 补充**：新增多种使用场景演示

### Switch

- **类型约束强化**：使用 `satisfies Record<keyof SwitchProps, any>` 模式
- **样式 Token 化**：迁移到 CSS 变量系统

### Slider

- **类型约束强化**：规范 props 定义与类型安全
- **样式 Token 化**：统一使用设计 Token 控制样式
- **边界防御增强**：添加数值范围校验和异常处理

### Upload

- **类型约束强化**：使用 `satisfies` 模式确保类型一致性
- **图标配置增强**：支持更灵活的图标自定义
- **国际化支持**：完善多语言文本处理

## 📊 质量提升

本次重构覆盖 **7 个组件**，系统性提升了代码质量：

- ✅ **类型安全**：全部组件使用 `satisfies Record<keyof XProps, any>` 模式，杜绝运行时与类型定义双源头漂移
- ✅ **样式规范**：硬编码样式迁移到 CSS 变量，支持主题定制
- ✅ **文档完善**：扩充 Demo 覆盖，提升使用体验

## 🔨 技术改进

- **类型安全模式统一**：所有重构组件采用 `satisfies Record<keyof XProps, any>` 约束，确保 props 对象与 TypeScript 接口完全一致
- **样式系统统一**：通过 CSS 变量实现样式 Token 化，提升主题定制能力
- **边界处理增强**：添加输入校验和异常处理，提升组件健壮性
