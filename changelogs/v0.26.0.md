# v0.26.0 (2026-07-18)

## ✨ 新特性

- **Tree**: 实现 `showLine` 连接线，全面对齐 AntD v6
- **Watermark**: 新增 `inherit` 属性，支持水印继承层级控制
- **Notification**: 新增 `showProgress` 进度条与 `rtl` 方向支持

## 🔧 重构

- **PopConfirm**: 规范 props 约束并 token 化样式
- **Message**: token 化硬编码样式并补充设计 Token 文档
- **Alert**: 规范 props 约束、事件类型导出并 token 化样式
- **Statistic**: 规范 props 约束、修复事件双触发并 token 化样式
- **Result**: 规范 props 约束、样式 token 化并补充 demo
- **Spin**: 规范 props 约束、样式 token 化并补充 demo
- **Timeline**: 类名与语义化 API 对齐 AntD v6，修复交替布局；图标改用 spin/rotate props
- **Tour**: 修正工具函数导入路径为相对引用

## 🐛 修复

- **Progress**: 修复圆形/仪表盘颜色被样式表覆盖，规范化 props

## 🔨 构建

- ESLint 迁移至 v10 flat config
- 降级 unplugin-vue-markdown 至 v29.2.0 修复类型检查错误
- 升级开发依赖，更新 @hmfw/icons 至 1.1.2
