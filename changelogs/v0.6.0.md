## [0.6.0] - 2026-07-02

### 💥 破坏性变更

- **移除图标重导出**：删除 `components/icon/` 目录，不再从主包 `@hmfw/ant-design` 重导出图标。用户需直接从 `@hmfw/icons` 导入图标和图标搜索 API
  ```typescript
  // 旧方式（已废弃）
  import { SearchOutlined, searchIcons } from '@hmfw/ant-design'

  // 新方式
  import { SearchOutlined, searchIcons } from '@hmfw/icons'
  ```
